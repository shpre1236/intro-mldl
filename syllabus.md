# Introduction to Machine Learning / Deep Learning

## Course Overview

본 클래스에서는 고등학교 수학을 넘어서는 개념은 과감히 생략하고 머신러닝과 딥러닝의 개념을 이해할 수 있도록 도와주는 총 5강으로 구성된 클래스입니다.

### Objective
- 기존 방법론과 머신러닝, 딥러닝의 차이를 안다.
- pytorch를 사용할 수 있다.

### Prerequisite
- 고등학교 수학
- CSED101 전산 이수

### Requirements
- 버벅이는 영어를 듣고 참아낼수 있는 인내심
- 강의 종료 후 피드백
- 강의 당 최소 질문 2개
